This source code contains 
main.c
server.c
client.c
headers.h
Makefile

How to run ?

download source code, unzip using command

$ unzip simple_udp_socket-master.zip

$ cd simple_udp_socket

$ ls
client.c  headers.h  main.c  Makefile  server.c

$ make 
$ make all
$ ls
client.c  headers.h  main.c  Makefile  run  server.c

run executable is generated now you can open two different terminal and run 
same executable file both sides. 
	In one terminal Choose option server and other terminal choose client.
